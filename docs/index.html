sobre mim

Me chamo Arthur, tenho 15 anos e sou fanático por tecnologia e programação. Estudo desenvolvimento de software, principalmente front-end

Há mais de um ano, comecei a estudar Python pelo Curso em Vídeo, do Gustavo Guanabara. Adorava usar estruturas lógicas simples para mostrar um resultado no terminal. Mas cade a interface? Ninguém gosta de telas pretas! Por isso, iniciei o curso de HTML e CSS da mesma plataforma.

Daí pra frente, foi pura aprendizagem: media querries, layout Flexbox, Grid, nomenclatura BEM, descobri como manipular elementos com Javascript e Jquery...

Atualmente estudo React e Frameworks CSS, como o Bootstrap.
