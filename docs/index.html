sobre mim

Me chamo Arthur, tenho 17 anos e sou fanático por tecnologia e programação.

Curso Desenvolvimento de Sistemas na Escola Técnica Estadual.

Há mais de três anos, comecei a estudar Python pelo Curso em Vídeo, do Gustavo Guanabara. Adorava usar estruturas lógicas simples para mostrar um resultado no terminal. Mas cadê a interface? Ninguém gosta de telas pretas! Por isso, iniciei o curso de HTML e CSS da mesma plataforma.

Daí pra frente, foi pura aprendizagem: media querries, layout Flexbox, Grid, nomenclatura BEM, descobri como manipular elementos com Javascript e Jquery...

Atualmente estudo React e Frameworks CSS, como o Bootstrap. Além disso, tenho projetos mobile feitos em React Native e APIs em Node.js (principalmente feitas com Express.js).
